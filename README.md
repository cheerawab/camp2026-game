# SITCON camp 2026 點數系統

# 遊戲規則說明

## 遊戲簡介

在活動中取得不同種類的小石、素材與開源力，並用這些資源組成自己的小石隊伍，參加知識王戰鬥、挑戰世界魔王、裝飾小隊基地。學員在完成幾種「我們激勵」的行為時，會得到對應類型的「小石」或道具。營隊結束時得到一個活動回顧紀錄。

---

## 核心循環

遊戲的主要循環如下：

1. 完成營隊實體任務
2. 獲得素材、開源力、特殊小石或強化道具
3. 從背包中選擇小石組成隊伍
4. 參加知識王戰鬥
5. 獲得佈景、道具、故事、強化資源
6. 強化小石或貢獻小隊基地
7. 挑戰世界任務與世界魔王
8. 解鎖新的故事、小石與遊戲回顧

## 重要名詞
### 開源力

開源力是遊戲中的主要通貨。

玩家可以透過完成活動任務、參加知識王、貢獻小隊或完成世界任務獲得開源力。

開源力可以用來：

- 強化小石
- 合成小石
- 升級小隊加成
- 購買特殊道具
- 裝飾小隊基地
- 解鎖小隊加乘
- 提升世界魔王戰鬥效果
- 解鎖小隊故事或小隊回顧

### 素材

素材是強化、合成與解鎖道具時使用的資源。

玩家可以透過以下方式取得素材：

- 完成實體任務掃 QR code 獲得
- 參加知識王戰鬥
- 完成世界任務

素材可以用來：

- 強化小石
- 合成小石
- 製作特殊道具
- 解鎖故事

### 知識王戰鬥

知識王戰鬥主要戰鬥形式。透過回答問題來增加隊伍戰鬥力，

知識王戰鬥會考驗：

- 課程理解
- 活動知識
- 資訊素養
- 社群知識
- 開源相關概念
- SITCON 活動內容

隊內小石會提供基礎數值加乘，但不能完全取代玩家答題。

### 世界魔王

世界魔王是所有玩家共同挑戰的大型事件。

世界魔王由全體玩家透過知識王戰鬥共同累積傷害。

挑戰世界魔王可以獲得：

- 大量強化道具
- 特殊素材
- 限定小石
- 小隊開源力
- 新故事
- 活動回顧紀錄

## 遊戲機制

### 知識王戰鬥
可以和 CP 或玩家面對面戰鬥，CP 會全部答對。

#### 戰鬥流程

- 選定五隻小石
- 回答問題（題目數量是 5 的倍數，可以挑選要玩幾題）
- 開始回答問題，根據難度獲得分數
- 根據雙方隊伍順序的小石種族進行加分
- 比較最終分數

#### 種族剋制
- 克制：工作力 × 1.25
- 被克制：abs(對方小石工作力-我方小石工作力) × -1.25
- 同種族：工作力 × 1.0
- 無克制關係：工作力 × 1.0
---

#### 戰鬥結果

- 答對越多題，獎勵越好
- 連續答對可以獲得額外加成
- 小石數值可以提高獎勵倍率
- 小石技能可以提供特殊效果
- 小石加乘基本上不會壓過玩家知識表現

---

#### 戰鬥獎勵

知識王戰鬥獲得：

- 素材
- 強化道具
- 佈景
- 小石合成材料
- 小隊開源力
- 世界魔王貢入場卷
- 活動回顧紀錄

### 賓果任務

玩家可以透過完成營隊中的實體任務取得遊戲資源。

實體任務包含：

- 參與社群攤位
- 與攤主互動
- 提問
- 回答問題
- 協助隊友
- 跨隊合作
- 參加活動
- 完成回饋表單
- 分享學習心得
- 分享解法或除錯過程

完成實體任務後，玩家可以獲得：

- 素材
- 開源力
- 特殊用途小石
- 強化道具
- 小隊開源力
- 解鎖小石合成操作的資格

部分任務需要 QR code、攤主確認、工作人員確認或表單紀錄。

### 小石合成

玩家可以使用素材、小石與特殊道具進行合成。

合成可以產生：

- 更高等級的小石
- 特殊型態小石
- 稀有小石
- 功能型小石
- 世界任務專用小石

合成是遊戲後期（圖鑑全破）解鎖的玩法。

合成需要數個造型、合成次數一樣的小石疊加為一隻小石，第一次合成需要兩隻一樣的小石，第二次要兩隻，第三次要三隻，以此類推。合成後造型不會變，用星星註記合成次數，合成後數值為原本的 1.3 倍。

### 小隊基地規則

每個小隊擁有自己的基地。基地代表小隊在營隊中的共同空間，可以被裝飾、升級，並留下小隊共同完成任務的痕跡。


#### 基地貢獻規則

玩家可以把部分開源力、素材或基地道具貢獻給小隊基地，強化中央裝置、裝飾基地或解鎖特定佈景。

- 裝飾基地
- 解鎖小隊加乘
- 強化小石隊伍基礎數值
- 提升世界魔王戰鬥後的道具取得
- 解鎖小隊故事
- 留下小隊活動回憶

小隊基地的成長來自全隊共同貢獻。

#### 基地佈景槽位

基地可以由中央裝置與多個佈景槽位組成。中央裝置負責小石隊伍的基礎數值強化；其他佈景槽位負責裝飾、掉落、回顧與活動收益，不再強化小石能力。

| 槽位 | 可裝備道具數 | 功能 |
| --- | ---: | --- |
| 中央裝置 | 1 | 使用開源力強化，直接提升小石隊伍基礎數值 |
| 小石休息區 | 6 | 顯示小石活動、休息與準備狀態，可以放較多素材或支援型道具 |
| 佈告欄 | 3 | 顯示小隊任務、共同目標與紀錄，適合任務與合作型道具 |
| 補給角落 | 3 | 放置素材箱、飲料、工具或神秘物資，適合資源與補給型道具 |
| 燈光與氣氛 | 2 | 改變基地色調、特效或活動狀態，適合活動與互動型道具 |
| 紀念展示架 | 7 | 展示活動限定道具與小隊成就，適合收藏、回顧與紀念型道具 |

基地背景是整體外觀，不計入上述裝備數量。背景可以用來呈現小隊風格、活動階段或回顧畫面，但不直接提供主要數值加成。

#### 中央裝置強化規則

中央裝置是小隊基地中唯一會直接強化小石隊伍數值的項目。

玩家可以把開源力投入中央裝置。中央裝置強化後，會直接提升小石隊伍的基礎數值，從 +1% 開始，理論上沒有上限。

- +1% 起跳 >
  -  讓小隊很早就感受到共同投資有回饋 
- 無上限 
  - 允許小隊長期投入開源力，形成共同成長感 
- 成本遞增
  - 每次提升需要更多開源力，避免前期快速拉開差距 |
- 只影響小石隊伍基礎數值
  - 不直接提高道具掉落、回顧權重或特定任務獎勵 

中央裝置的定位是「小隊共同灌注開源力的核心」。它可以很強，但需要大量共同投入；裝飾道具則負責改變小隊擅長取得什麼資源。

#### 裝飾道具效果規則

裝飾道具放上基地後，決定小隊更容易取得哪一類獎勵。裝飾道具不強化小石能力，也不直接提高小石隊伍基礎數值。

| 效果類型 | 說明 | 例子 |
| --- | --- | --- |
| 魔王戰鬥掉落 | 提升世界魔王戰鬥後取得道具的品質或數量 | 魔王戰鬥獲得道具 2 倍 |
| 指定類型道具掉落 | 讓某種類型的道具更容易出現 | 強化素材、合成素材、消耗道具更容易掉落 |
| 任務獎勵偏向 | 讓特定任務更容易給出指定獎勵 | 攤位任務更容易給探索型素材 |
| 活動回顧偏向 | 讓特定回顧片段或紀念內容更容易被記錄 | 晚會、CP 互動、世界魔王回顧更容易出現 |
| 裝飾與收藏 | 純外觀、展示或紀念用途 | 展示活動限定道具、小隊共同成就 |

例如裝備「世界魔王作戰圖」時，可以讓世界魔王戰鬥後獲得道具 2 倍；裝備「小隊補給箱」時，可以讓素材類道具更容易掉落。

多個裝飾道具可以指向同一類掉落，但同類效果需要有觸發限制、每日上限或結算上限，避免小隊只堆同一種道具造成失衡。

| 晚會氣氛燈串 | 燈光與氣氛 | 晚會任務更容易給娛樂型素材 | 只要一亮起來，就有人開始想排節目。 |
| 深夜除錯燈 | 燈光與氣氛 | 技術任務更容易掉落限定道具 | 一開燈就會有人說「我再試最後一次」。 |
| 掌聲音效盒 | 燈光與氣氛 | 活動完成任務更容易給開源力 | 按下去會出現掌聲，適合給完成任務的人一點場面。 |
| 貼紙牆一號機 | 紀念展示架 | 活動回顧素材更容易出現 | 貼滿了不知道誰拿來的貼紙，已經看不出原本牆色。 |
| 貼紙風暴展示牆 | 紀念展示架 | 活動限定任務更容易給限定佈景碎片 | 牆已經不是牆，是一段彩色的營隊歷史。 |
| 世界魔王紀念核心 | 紀念展示架 | 魔王階段結算更容易出現限定道具 | 據說靠近它會聽到大家一起喊「再打一場」。 |
| 小隊名牌收藏盒 | 紀念展示架 | 小隊回顧片段更容易出現 | 收著各種手寫名牌，其中幾張字跡已經變成謎題。 |
| CP 簽名板 | 紀念展示架 | CP 互動任務更容易給紀念道具 | 簽名旁邊通常會附一個看不懂但很有氣勢的圖案。 |
| 最後一晚拍立得牆 | 紀念展示架 | 活動結束回顧更容易出現限定片段 | 不是最強的道具，但很適合讓人突然安靜一下。 |


#### 備註

- 中央裝置只能透過開源力強化
- 裝飾道具不強化小石能力，只影響掉落、回顧、素材或道具取得方向
- 佈景解鎖可以消耗素材或同槽位的佈景碎片
- 小隊貢獻紀錄進入活動回顧，但不做成公開排行榜

## 小隊加乘

小隊基地升級後，可以提供小隊加乘。

小隊加乘包含：

- 知識王獎勵提升
- 世界魔王貢獻提升
- 合成消耗降低
- 特殊道具掉落率提升
- 小隊佈景解鎖

## 任務

賓果蓋章連線，會有多張賓果卡片對應每日任務、常駐任務、活動限定任務，賓果卡上每一格都是一個任務，單一擊破、連線都發獎勵。


> 活動限定任務只有活動結束在結算時才發獎勵

任務分為以下幾個種類：


- 培育出某種小石
- 填寫表單、活動回饋
- 戰鬥任務
  - 答對 X 種分類的題目
  - 參加幾場知識王決鬥
  - 個人對魔王造成的傷害
- 搜集指定開源力

---

## 世界魔王規則

世界魔王是大型共同挑戰，所有人共用魔王的血量，

玩家透過知識王戰鬥對世界魔王造成貢獻，傷害就是知識王戰鬥拿到的。

世界魔王可以分成多個階段：


世界魔王獎勵包含：

- 大量強化道具
- 限定素材
- 限定小石
- 限定佈景
- 開源力

---

## 佈景與道具

玩家可以透過知識王戰鬥、世界任務、實體任務或世界魔王取得佈景與道具。

佈景可以用於：

- 裝飾個人頁面
- 裝飾小隊基地
- 顯示活動紀念
- 呈現玩家風格

道具可以用於：

- 強化小石
- 合成小石
- 提升知識王收益
- 解鎖故事
- 支援小隊基地
- 挑戰世界任務

## 遊戲回顧

活動結束時，系統會為玩家產生遊戲行為回顧。

回顧內容包含：

- 參加了哪些知識王戰鬥
- 搜集了哪些小石
- 貢獻了多少小隊開源力
- 參與了哪些世界任務
- 對世界魔王造成多少貢獻
- 解鎖了哪些故事與佈景
- 最常使用的小石隊伍
- 最具代表性的活動行為

- 誰貢獻最多


# 角色設計

小石造型可以作為任務獎勵、合成目標與限定收藏。每隻小石仍然歸屬於五大種族之一，但外觀、入手方式與故事定位不同。

| 小石名稱 | 種族 | 造型設計 | 如何入手 |
| --- | --- | --- | --- |
| 地圖小石 | 探索型小石 | 背著小背包，頭上插著摺疊地圖，走路時會東張西望。 | 完成「營地探索」任務，或用探索碎片 + 攤位貼紙合成。 |
| 攤位小石 | 探索型小石 | 身上掛滿貼紙與小徽章，手上拿著集章卡。 | 參觀指定數量攤位後獲得，或用攤位貼紙 + 問答印章合成。 |
| 雷達小石 | 探索型小石 | 頭頂有小天線，靠近隱藏任務時會發出微光。 | 完成隱藏 QR code 任務，或用地圖小石 + 感應零件合成。 |
| 問號小石 | 靈光型小石 | 頭上漂著一個小問號，思考時眼睛會亮起來。 | 在課程中提出問題，或用筆記紙片 + 靈感碎片合成。 |
