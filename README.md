# SITCON Camp 2026 點數系統

## 遊戲定位

本系統是營隊期間使用的知識探索遊戲。

遊戲以知識王戰為核心互動，用小石記錄玩家實際做過的行為，用任務引導學員參與現場活動，用開源力與道具支撐收藏合成，用世界魔王呈現全營共同進度，最後在 Day 5 產生活動回顧與不同類型成就。

核心原則：

- 答題重點是理解，不是刷分。
- 小石重點是對應玩家實際行為、收藏與題色加成，不是戰力。
- 開源力重點是兌換與合成，不是養成壓力。
- 主頁就是基地，重點是個人與小隊回憶展示，不是數值成長。
- 結算重點是多元成就，不是單一排行榜。

---

## 核心循環

玩家的主要遊戲流程如下：

1. 參與營隊活動、完成任務或進行知識王戰。
2. 獲得開源力、小石、造型道具、基地佈景或回顧紀錄。
3. 使用開源力兌換收藏內容，或用小石與道具合成新造型。
4. 透過知識王戰答題、解析與訂正累積學習紀錄。
5. 透過全營行動推進世界魔王共同進度。
6. 在 Day 5 依照參與、學習、合作、探索與回顧產生不同類型成就。

## 重要名詞

### 開源力

開源力是遊戲中的主要貨幣。

玩家可以透過以下方式取得開源力：

- 完成營隊任務
- 參加知識王戰
- 完成知識王戰訂正
- 參與世界魔王階段
- 完成小隊或現場活動

開源力可以用來：

- 兌換基地佈景
- 兌換小石造型道具
- 進行小石造型合成
- 解鎖活動回顧展示效果
- 兌換世界魔王階段紀念品

### 知識王戰

知識王戰是遊戲的主要答題玩法。玩家透過回答題目取得分數、開源力、道具、題組紀錄與訂正紀錄。

每場知識王戰會使用一組題目。題目本身不固定顏色，而是在生成知識王戰題組時，才為每道題目產生一個題色。題色對應五種小石類型。玩家在戰前排好 5 隻小石，題目依序對應小石位置；如果題色與該位置小石類型相同，且玩家答對，就獲得題色加成。

知識王戰包含：

- CP 戰：練習與訂正模式。
- 面對面對戰：兩位玩家現場互相挑戰。
- 世界魔王：特殊的知識王戰，全營共同推進。

### 世界魔王

世界魔王是特殊的知識王戰。用知識王戰的答題、題色與小石位置規則，全營共同擁有魔王血條。

世界魔王的題目會對應當天上課內容，並依照當日課程主題調整題色生成權重，提高特定題色的出現機率。例如當天課程偏工程實作，藍色題可以有較高機率出現；若當天活動偏社群互動，紫色題或綠色題可以有較高機率出現。

所有玩家共同推進同一條世界魔王階段進度。共鬥勝利後，全體參與者可以獲得限定造型小石或其他紀念獎勵。

### 題目與題色

資料庫中的每筆題目都是單純的題目資料，包含題幹、選項、正解、解析、主題、難度等內容。題目在資料庫中不固定屬於任何一種小石顏色。

題色是知識王戰生成題組時附加到題目上的戰鬥屬性。一般知識王戰可以平均或依活動需求生成題色；世界魔王則會依當天課程主題調整題色權重。

因此，同一道資料庫題目在不同場知識王戰中，可以被生成為不同題色。

### 小石

小石是玩家收集、展示與帶入知識王戰的主要角色，也代表玩家在營隊中實際做過的行為。

小石有五種類型，每種類型對應一種營隊行為與一種題色。玩家透過提問、互動、協助、解題、參與活動等行為取得對應的小石。

小石在知識王戰中的作用是題色加成。玩家在對戰前排好 5 隻小石，五隻小石依序代表第 1 到第 5 題的答題位置。每題生成後會有一種題色；若該題題色剛好對上該位置的小石類型，且玩家答對該題，則獲得小幅加分。

小石不提供工作力、種族剋制、掉落率或額外答案提示。小石加成只在答對時生效，答錯不加分。

| 探索型小石 | 綠色題 | `#2EAD72` | `#DDF7EA` | `#0B6B3A` | 地圖、攤位、發現 | 逛攤位、問問題、參與社群事件 |
| 靈光型小石 | 黃色題 | `#F2B705` | `#FFF3BF` | `#7A5200` | 靈感、課程、思考 | 提問、回答、做筆記、完成反思 |
| 共鳴型小石 | 紫色題 | `#8E6AD8` | `#EFE7FF` | `#4B2E83` | 小隊、連結、合作 | 幫助隊友、跨隊合作、社交互動 |

小石可以用於：

- 知識王戰題色加成
- 基地展示
- 造型合成
- 活動回顧紀錄

### 道具

道具是任務、知識王戰、世界魔王或活動階段給予的收藏與合成材料。


基本規則：

- 道具不能彼此合成。
- 合成配方必須包含至少一隻小石。
- 合成結果是新造型、限定小石或展示效果。
- 合成不會提高工作力、戰鬥勝率或掉落率；造型仍只依照小石類型參與題色加成。

## 知識王戰

知識王戰是遊戲的核心玩法，題目範圍包含課程理解、資訊素養、社群知識、開源概念、SITCON 活動內容與營隊現場事件。

每道資料庫題目在生成知識王戰題組時，會獲得一個題色，對應五種小石類型。每題只能作答一次，不允許重答。答題後一定顯示正解與解析。

### 小石隊伍與題色加成

玩家進入知識王戰前，選擇 5 隻小石並排好順序。這 5 隻小石代表 5 個答題位置：

- 第 1 題對應第 1 隻小石。
- 第 2 題對應第 2 隻小石。
- 第 3 題對應第 3 隻小石。
- 第 4 題對應第 4 隻小石。
- 第 5 題對應第 5 隻小石。

若題數超過 5 題，從第 6 題開始循環回第 1 隻小石。也就是第 N 題對應第 `((N - 1) % 5) + 1` 隻小石。

計分規則：

- 答對題目：獲得基礎分。
- 答對且題色對上該位置小石類型：獲得題色加成。
- 答錯：不獲得基礎分，也不獲得題色加成。
- 訂正：不改變原本分數，只留下訂正紀錄與少量獎勵。

題色加成的目的，是讓小石隊伍有排陣與押題的遊戲感，但不引入工作力、剋制或複雜倍率。

### 基本答題流程

1. 玩家進入題目。
2. 系統顯示題目、選項與必要提示。
3. 玩家作答。
4. 系統顯示正解與解析。
5. 答對者獲得完整題目獎勵。
6. 答錯者不重答，但可以進入訂正區。

### 訂正區

訂正區採最輕量設計，用來讓答錯不只是失分，而是留下學習紀錄。

答錯後玩家可以看解析，並完成一個簡短訂正動作。第一版可以採用按鈕或錯因選擇。

可用錯因：

- 我看錯題目
- 我不熟這個概念
- 我被選項混淆
- 我是猜的
- 我看完解析理解了

完成訂正後可以獲得：

- 少量開源力
- 訂正紀錄
- 活動回顧標籤
- 世界魔王共同進度

訂正不等於重答，不會改變原本題目的答題結果。

### CP 戰

CP 戰是知識王戰中的練習與訂正模式。

CP 預設會全部答對。玩家與 CP 一起進行題目時，重點不是公平競技，而是透過答題、解析與訂正取得學習紀錄。

CP 戰可以給予：

- 題目獎勵
- 訂正獎勵
- 少量開源力
- 題組完成紀錄
- 回顧標籤

### 面對面對戰

面對面對戰是兩位玩家現場互相挑戰的知識王戰模式。

規則方向：

- 雙方回答同一組題目。
- 勝負主要看答對題數。
- 若答對題數相同，可以用剩餘時間或加題決勝。
- 小石依照答題位置提供題色加成，不提供工作力、剋制或單方答案提示。
- 輸家仍可獲得參與獎勵與訂正獎勵。
- 不做連勝高倍率。

面對面對戰的目標是創造現場互動與刺激感，不是製造壓力或數值差距。

### 知識王戰獎勵

知識王戰可以獲得：

- 開源力
- 造型道具
- 題組完成紀錄
- 小石
- 基地佈景
- 活動回顧標籤
- 世界魔王共同進度

## 任務

任務用來引導學員參與營隊現場，而不是要求玩家刷表格。

任務可以包含：

- 參與社群攤位
- 與攤主互動
- 在課程中提問
- 回答問題
- 協助隊友
- 跨隊合作
- 參加晚會或團康
- 完成回饋表單
- 分享學習心得
- 分享解法或除錯過程

任務獎勵可以包含：

- 開源力
- 小石
- 造型道具
- 基地佈景
- 世界魔王共同進度
- 活動回顧標籤

第一版任務應以每日少量、高品質、低驗證負擔為原則。

> 小石 + 指定道具 = 新造型小石或限定收藏

## 基地主頁

主頁就是基地，是玩家進入系統後看到的主要展示空間。基地同時呈現個人的小石收藏、實際行為紀錄，以及小隊共同解鎖的回憶。

透過任務取得基地佈景主題，不做自由裝飾、不做基地升級、不做站崗位置管理。

小組同一時間只能配置一種基地佈景。佈景是小組統一的設定；小組更換佈景後，所有小組成員參加知識王戰都套用該佈景效果，包含 CP 戰、面對面對戰與世界魔王。

佈景沒有升級線。取得時效果是多少，就是固定多少；不能投入開源力升級，也不能疊加多個佈景效果。

基地佈景可以來自：

- 小隊任務
- 每日活動
- 攤位互動
- 世界魔王階段
- 晚會或團康
- Day 5 回顧

基地佈景會影響主頁展示，也會在知識王戰中提供簡單效果。佈景效果只分成少數幾類，避免變成複雜裝備系統。

佈景效果類型：

- 基礎分增加：所有知識王戰中，答對題目時獲得的基礎分增加固定值。
- 自動染色 3 題：戰鬥中若當前題色沒有對應該位置的小石，自動將該題染成對應小石的題色，最多 3 題。
- 自動染色 5 題：戰鬥中若當前題色沒有對應該位置的小石，自動將該題染成對應小石的題色，最多 5 題。
- 戰後開源力提升：知識王戰結算後，獲得開源力提升固定比例。
- 合成道具掉落率提升：知識王戰結算後，小石合成道具掉落率提升固定比例。

自動染色規則：

- 只有題色沒有對應該位置小石時才會觸發。
- 若題色本來就對應該位置小石，不消耗染色次數。
- 染色只改變該題的題色，不改變題目內容、正解或解析。
- 染色後仍然必須答對，才會取得基礎分與題色加成。

佈景範例：

| 佈景 | 取得方向 | 戰鬥效果 | 展示意義 |
| --- | --- | --- | --- |
| 攤位地圖 | 完成攤位探索任務 | 自動染色 3 題 | 小隊一起探索社群攤位 |
| 靈感白板 | 完成課程提問或反思任務 | 自動染色 3 題 | 小隊留下學習與討論痕跡 |
| 共鳴留言板 | 完成隊友協助任務 | 戰後開源力提升 | 小隊合作與互相支持 |
| 深夜除錯 | 完成技術任務或除錯協助 | 自動染色 5 題 | 小隊的工程精神 |
| 晚會舞台 | 參與晚會或活動任務 | 戰後開源力提升 | 小隊的活動能量 |
| 世界魔王作戰圖 | 推進世界魔王階段 | 所有知識王戰基礎分增加 | 全營共同事件紀念 |
| 最後一晚拍立得牆 | Day 5 結算 | 合成道具掉落率提升 | 營隊回憶收束 |

## 世界魔王

世界魔王是特殊的知識王戰，也是全營共同進度，不是個人輸出競賽。

世界魔王沿用知識王戰的答題、題色與小石位置規則。所有玩家共同推進同一條世界魔王階段進度。系統可以記錄個人與小隊曾參與哪些階段，但不做個人傷害榜。

世界魔王題目會針對當天上課內容出題，讓世界魔王成為每日課程的共同複習與收束。題色會在題組生成時依當日課程主題調整權重，讓特定題色有較高機率出現。例如 Day 3 若主軸是 AI / ML，魔王題組就以當天相關概念、講師提到的重點與延伸理解為主，並可以提高黃色題或藍色題的出現機率。

世界魔王進度來源可以包含：

- 完成知識王戰題目
- 答對題目
- 閱讀解析
- 完成訂正
- 完成主題題組
- 完成營隊任務
- 參與世界魔王限定活動

世界魔王階段獎勵可以包含：

- 共鬥勝利後，全體參與者獲得限定造型小石
- 限定造型道具
- 基地佈景
- 活動回顧標籤
- 開源力

世界魔王的目標是讓全營看到共同推進的事件感，而不是讓少數高分玩家壟斷成果。

## 活動回顧與 Day 5 結算

活動結束時，系統會為玩家產生活動回顧。

個人回顧可以包含：

- 完成了哪些任務
- 參加了哪些知識王戰
- 答對哪些主題
- 完成哪些訂正
- 收集哪些小石
- 合成哪些造型
- 解鎖哪些基地佈景
- 參與哪些世界魔王階段
- 最具代表性的營隊行為

Day 5 不只獎勵最高分，而是依照不同類型成就發放實體獎勵或紀念。

成就方向：

| 成就類型 | 判斷方向 |
| --- | --- |
| 知識探索獎 | 完成多個主題題組、穩定參與知識王戰 |
| 訂正成長獎 | 答錯後完成訂正、留下學習軌跡 |
| 小隊共鳴獎 | 完成小隊任務、協助隊友、參與合作活動 |
| 社群探索獎 | 攤位互動、跨隊交流、主動提問 |
| 工程精神獎 | 技術任務、除錯、分享解法 |
| 活動能量獎 | 晚會、團康、現場活動參與 |
| 世界魔王參與獎 | 參與全營共同進度與階段推進 |
| 收藏達成獎 | 收集小石、造型或基地佈景 |
