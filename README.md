# SITCON Camp 2026 點數系統

## 遊戲定位

本系統是營隊期間使用的知識探索遊戲。

遊戲以知識王答題為核心互動，用小石記錄玩家實際做過的行為，用任務引導學員參與現場活動，用開源力與道具支撐收藏合成，用世界魔王呈現全營共同進度，最後在 Day 5 產生活動回顧與不同類型成就。

核心原則：

- 答題重點是理解，不是刷分。
- 小石重點是對應玩家實際行為、收藏與輕輔助，不是戰力。
- 開源力重點是兌換與合成，不是養成壓力。
- 主頁就是基地，重點是個人與小隊回憶展示，不是數值成長。
- 結算重點是多元成就，不是單一排行榜。

---

## 核心循環

玩家的主要遊戲流程如下：

1. 參與營隊活動、完成任務或進行知識王。
2. 獲得開源力、小石、造型道具、基地佈景或回顧紀錄。
3. 使用開源力兌換收藏內容，或用小石與道具合成新造型。
4. 透過知識王答題、解析與訂正累積學習紀錄。
5. 透過全營行動推進世界魔王共同進度。
6. 在 Day 5 依照參與、學習、合作、探索與回顧產生不同類型成就。

## 重要名詞

### 開源力

開源力是遊戲中的主要貨幣。

玩家可以透過以下方式取得開源力：

- 完成營隊任務
- 參加知識王
- 完成知識王訂正
- 參與世界魔王階段
- 完成小隊或現場活動

開源力可以用來：

- 兌換基地佈景
- 兌換小石造型道具
- 進行小石造型合成
- 解鎖活動回顧展示效果
- 兌換世界魔王階段紀念品

### 小石

小石是玩家收集、展示與帶入知識王的主要角色，也代表玩家在營隊中實際做過的行為。

小石有五種類型，每種類型對應一種營隊行為與輕量輔助。玩家不是靠抽到強角變強，而是透過提問、互動、協助、解題、參與活動等行為取得對應的小石。小石不提供工作力、攻擊力、分數倍率或掉落率加成。

| 類型 | 對應行為 | 取得方向 | 輕量輔助 |
| --- | --- | --- | --- |
| 探索型小石 | 地圖、攤位、發現 | 逛攤位、問問題、參與社群事件 | 顯示相關延伸題、延伸閱讀或任務提示 |
| 靈光型小石 | 靈感、課程、思考 | 提問、回答、做筆記、完成反思 | 在知識王中提供關鍵字提示或少量選項提示 |
| 共鳴型小石 | 小隊、連結、合作 | 幫助隊友、跨隊合作、社交互動 | 在小隊任務或合作活動中提供小幅進度輔助 |
| 工程型小石 | 程式、除錯、工具 | 修 bug、分享解法、完成技術任務 | 顯示概念拆解、術語說明或技術提示 |
| 娛樂型小石 | 晚會、活動、活力 | 參與晚會、團康、活動事件 | 強化結算演出、基地展示與活動回顧效果 |

小石可以用於：

- 知識王輕量輔助
- 基地展示
- 造型合成
- 活動回顧紀錄

### 道具

道具是任務、知識王、世界魔王或活動階段給予的收藏與合成材料。


基本規則：

- 道具不能彼此合成。
- 合成配方必須包含至少一隻小石。
- 合成結果是新造型、限定小石或展示效果。
- 合成不會提高答題分數、戰鬥勝率或掉落率。

## 知識王

知識王是遊戲的核心玩法，題目範圍包含課程理解、資訊素養、社群知識、開源概念、SITCON 活動內容與營隊現場事件。

每題只能作答一次，不允許重答。答題後一定顯示正解與解析。

### 基本答題流程

1. 玩家進入題目。
2. 系統顯示題目、選項與必要提示。
3. 玩家作答。
4. 系統顯示正解與解析。
5. 答對者獲得完整題目獎勵。
6. 答錯者不重答，但可以進入訂正區。

### 訂正區

訂正區採最輕量設計，用來讓答錯不只是失分，而是留下學習紀錄。

答錯後玩家可以看解析，並完成一個簡短訂正動作。第一版可以採用按鈕或錯因選擇。

可用錯因：

- 我看錯題目
- 我不熟這個概念
- 我被選項混淆
- 我是猜的
- 我看完解析理解了

完成訂正後可以獲得：

- 少量開源力
- 訂正紀錄
- 活動回顧標籤
- 世界魔王共同進度

訂正不等於重答，不會改變原本題目的答題結果。

### CP 戰

CP 戰是知識王中的練習與訂正模式。

CP 預設會全部答對。玩家與 CP 一起進行題目時，重點不是公平競技，而是透過答題、解析與訂正取得學習紀錄。

CP 戰可以給予：

- 題目獎勵
- 訂正獎勵
- 少量開源力
- 題組完成紀錄
- 回顧標籤

### 面對面對戰

面對面對戰是兩位玩家現場互相挑戰的知識王模式。

規則方向：

- 雙方回答同一組題目。
- 勝負主要看答對題數。
- 若答對題數相同，可以用剩餘時間或加題決勝。
- 小石只提供輕量輔助，不提供分數倍率。
- 輸家仍可獲得參與獎勵與訂正獎勵。
- 不做連勝高倍率。

面對面對戰的目標是創造現場互動與刺激感，不是製造壓力或數值差距。

### 知識王獎勵

知識王可以獲得：

- 開源力
- 造型道具
- 題組完成紀錄
- 小石
- 基地佈景
- 活動回顧標籤
- 世界魔王共同進度

## 任務

任務用來引導學員參與營隊現場，而不是要求玩家刷表格。

任務可以包含：

- 參與社群攤位
- 與攤主互動
- 在課程中提問
- 回答問題
- 協助隊友
- 跨隊合作
- 參加晚會或團康
- 完成回饋表單
- 分享學習心得
- 分享解法或除錯過程

任務獎勵可以包含：

- 開源力
- 小石
- 造型道具
- 基地佈景
- 世界魔王共同進度
- 活動回顧標籤

第一版任務應以每日少量、高品質、低驗證負擔為原則。

---

## 小石造型合成

合成保留為收藏與外觀玩法，不做數值養成。

基本公式：

> 小石 + 指定道具 = 新造型小石或限定收藏

合成範例：

| 配方 | 結果 | 定位 |
| --- | --- | --- |
| 探索型小石 + 攤位貼紙 | 攤位小石 | 攤位、集章、社群互動紀念 |
| 探索型小石 + 活動地圖 | 地圖小石 | 營地探索與隱藏任務紀念 |
| 靈光型小石 + 提問卡 | 問號小石 | 提問、思考與課程互動紀念 |
| 靈光型小石 + 課程筆記 | 筆記小石 | 筆記、反思與學習紀錄 |
| 共鳴型小石 + 感謝卡 | 握手小石 | 隊友協助與 CP 互動紀念 |
| 共鳴型小石 + 交換卡 | 連線小石 | 跨隊合作紀念 |
| 工程型小石 + Bug 標籤 | Bug 小石 | 回報問題與除錯紀念 |
| 工程型小石 + 程式碼卡 | 終端機小石 | 技術任務與解法分享紀念 |
| 娛樂型小石 + 活動票根 | 麥克風小石 | 晚會、舞台與活動參與紀念 |
| 娛樂型小石 + 彩帶 | 彩帶小石 | 團康與氣氛紀念 |
| 任一小石 + 回顧膠卷 | 回憶小石 | 活動回顧限定收藏 |
| 工程型小石 + 開源核心 | 開源核心小石 | 世界魔王階段紀念 |

延後項目：

- 小石升星
- 小石數值強化
- 小石互相合成
- 稀有度影響戰力
- 合成後提高答題分數

## 基地主頁

主頁就是基地，是玩家進入系統後看到的主要展示空間。基地同時呈現個人的小石收藏、實際行為紀錄，以及小隊共同解鎖的回憶。

第一版只開放透過任務取得基地佈景主題，不做自由裝飾、不做基地升級、不做站崗位置管理。

基地佈景可以來自：

- 小隊任務
- 每日活動
- 攤位互動
- 世界魔王階段
- 晚會或團康
- Day 5 回顧

基地佈景只影響主頁展示與回顧，不提供戰力、倍率、掉落率或知識王優勢。

佈景範例：

| 佈景 | 取得方向 | 展示意義 |
| --- | --- | --- |
| 攤位地圖 | 完成攤位探索任務 | 小隊一起探索社群攤位 |
| 靈感白板 | 完成課程提問或反思任務 | 小隊留下學習與討論痕跡 |
| 共鳴留言板 | 完成隊友協助任務 | 小隊合作與互相支持 |
| 深夜除錯 | 完成技術任務或除錯協助 | 小隊的工程精神 |
| 晚會舞台 | 參與晚會或活動任務 | 小隊的活動能量 |
| 世界魔王作戰圖 | 推進世界魔王階段 | 全營共同事件紀念 |
| 最後一晚拍立得牆 | Day 5 結算 | 營隊回憶收束 |

---

## 世界魔王

世界魔王是全營共同進度，不是個人輸出競賽。

所有玩家共同推進同一條世界魔王階段進度。系統可以記錄個人與小隊曾參與哪些階段，但不做個人傷害榜。

魔王戰題目會針對當天上課內容出題，讓世界魔王成為每日課程的共同複習與收束。例如 Day 3 若主軸是 AI / ML，魔王題組就以當天相關概念、講師提到的重點與延伸理解為主。

世界魔王進度來源可以包含：

- 完成知識王題目
- 答對題目
- 閱讀解析
- 完成訂正
- 完成主題題組
- 完成營隊任務
- 參與世界魔王限定活動

世界魔王階段獎勵可以包含：

- 共鬥勝利後，全體參與者獲得限定造型小石
- 限定造型道具
- 基地佈景
- 活動回顧標籤
- 開源力

世界魔王的目標是讓全營看到共同推進的事件感，而不是讓少數高分玩家壟斷成果。

---

## 活動回顧與 Day 5 結算

活動結束時，系統會為玩家產生活動回顧。

個人回顧可以包含：

- 完成了哪些任務
- 參加了哪些知識王
- 答對哪些主題
- 完成哪些訂正
- 收集哪些小石
- 合成哪些造型
- 解鎖哪些基地佈景
- 參與哪些世界魔王階段
- 最具代表性的營隊行為

Day 5 不只獎勵最高分，而是依照不同類型成就發放實體獎勵或紀念。

成就方向：

| 成就類型 | 判斷方向 |
| --- | --- |
| 知識探索獎 | 完成多個主題題組、穩定參與知識王 |
| 訂正成長獎 | 答錯後完成訂正、留下學習軌跡 |
| 小隊共鳴獎 | 完成小隊任務、協助隊友、參與合作活動 |
| 社群探索獎 | 攤位互動、跨隊交流、主動提問 |
| 工程精神獎 | 技術任務、除錯、分享解法 |
| 活動能量獎 | 晚會、團康、現場活動參與 |
| 世界魔王參與獎 | 參與全營共同進度與階段推進 |
| 收藏達成獎 | 收集小石、造型或基地佈景 |
