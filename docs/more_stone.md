# 活動限定小石

活動限定小石不走一般合成樹，主要作為特殊活動、劇情選擇、夜談參與或闖關活動的紀念獎勵。每顆小石仍然只有一個主要被動效果，但是效果會異常強力。

## 海森堡 bug 結局小石

### 秩序守護者小石

- 取得方式：海森堡 bug 選擇 A 結局「刪除證據，修補漏洞」。
- 定位：工程型活動限定小石。
- 設定：牠把破碎的帳本重新排齊，讓位元城看起來再次穩定。只是每一次修補完成後，牠都會多看一眼那些被抹去的名字。
- 特殊能力：`秩序校準`
- 效果：提高戰鬥分數。牠象徵穩定、修補與系統維護，適合給答題戰鬥中的穩定輸出加成。

### 自由反抗者小石

- 取得方式：海森堡 bug 選擇 B 結局「公開帳本，還原真相」。
- 定位：共鳴型活動限定小石。
- 設定：牠把被藏起來的帳本高高舉起，讓所有人看見系統背後的真相。即使城市因此短暫混亂，牠仍相信自由比沉默更接近答案。
- 特殊能力：`公開帳本`
- 效果：提高結算開源力。牠象徵揭露、分享與集體行動，適合在完成戰鬥或任務後給更多開源力回饋。

## 特殊造型小石

### 烏龜小石

- 取得方式：Python turtle 相關活動或特殊獎勵。
- 定位：靈光型活動限定小石。
- 設定：牠慢慢走，慢慢畫，把每一次轉彎都留成可以回看的線。答案不一定要最快出現，有時候一步一步畫出來，反而更清楚。
- 特殊能力：`烏龜刪法`
- 效果：戰鬥答題時，有高機率在最後 5 秒刪去兩個錯誤選項。觸發時可以用 Python turtle 風格的線條慢慢圈掉錯誤答案。

### 爐邊小石

- 取得方式：爐邊夜談活躍獎勵。
- 定位：共鳴型活動限定小石。
- 設定：牠總是在夜談時靠近火光，安靜聽完每個人的故事。有人願意開口時，牠身上的光就會亮一點。
- 特殊能力：`夜談餘火`
- 效果：提高結算開源力。牠鼓勵分享、傾聽與交流，適合給參與活動後的回饋加成。

## 闖關活動小石

### 指令盲旅小石

- 取得方式：完成或在「指令盲旅」關卡表現突出。
- 定位：工程型活動限定小石。
- 設定：牠戴上眼罩後不急著往前衝，而是等待隊友把混亂變成一套清楚的指令。每一次成功抵達，都是大家共同定義出來的路。
- 特殊能力：`指令校準`
- 效果：提高戰鬥分數。牠代表清楚定義、穩定執行與團隊協作。

### 人類大語言模型小石

- 取得方式：完成或在「人類大語言模型」關卡表現突出。
- 定位：靈光型活動限定小石。
- 設定：牠只能一次留下短短幾個字，卻努力把前一個人的意思接住。當句子一路傳到最後，牠才發現理解本身也需要團隊合作。
- 特殊能力：`語意接龍`
- 效果：戰鬥答題時，有機率刪去一個錯誤選項。牠象徵從片段中抓出方向，適合做答題輔助。

### 馮紐曼小石

- 取得方式：完成或在「馮紐曼架構」關卡表現突出。
- 定位：工程型活動限定小石。
- 設定：牠把輸入、記憶、計算和輸出分成不同位置，再把散落的偏旁重新組回答案。看似混亂的流程，其實有一套安靜運轉的架構。
- 特殊能力：`架構運轉`
- 效果：提高戰鬥分數。牠代表資料流、記憶體、CPU 與輸出的協作流程。

### 封包救援小石

- 取得方式：完成或在「封包掉包」關卡表現突出。
- 定位：工程型活動限定小石。
- 設定：牠在錯亂與遺失的訊息之間奔跑，把破碎的封包一段一段撿回來。只要隊友還願意重傳，答案就不會真的消失。
- 特殊能力：`要求重傳`
- 效果：提高戰鬥分數。牠象徵錯誤修復、封包重傳與從混亂中還原答案。

### Prompt Injection 小石

- 取得方式：完成或在 Prompt Injection 相關關卡表現突出。
- 定位：靈光型活動限定小石。
- 設定：牠最擅長從看似正常的句子裡找到奇怪的入口。面對被偷走方向的提示詞，牠會先停下來問一句：這真的是我們要回答的問題嗎？
- 特殊能力：`提示防火牆`
- 效果：戰鬥答題時，有機率刪去一個錯誤選項。牠代表辨識干擾、排除錯誤方向與守住原本問題。
